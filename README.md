# ClipboardCopy.js

## Setup

```html
<script src="clipboard-copy.js"></script>
```

## Usage

```HTML
<button type="button" class="copy" data-copy="Copy to clipboard.">
    copy
</button>

<script src="clipboard-copy.js"></script>
<script>
new ClipBoardCopyJS('.btn');
</script>
```
